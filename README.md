1️⃣ What is the difference between var, let, and const?

answer:
i. var => function scoped, can be redeclared and updated, hoisted
ii.let => Block scoped , can be updated but not redeclared

iii. const => Block scoped can not be updated or redeclared

2️⃣ What is the spread operator (...)?

answer: Expands arrays or objects into individual elements

3️⃣ What is the difference between map(), filter(), and forEach()?

answer:
i. map() => returns a new array after applying a function

ii. filter() => returns a new array with elements that pass a condition

iii. foreach() => loops through array but returns undefined

4️⃣ What is an arrow function?

answer:
Shorter function syntax using => .
example: const sum= (a,b) => a+b;

5️⃣ What are template literals?

answer:
Strings using backticks ` with ${} for variables.
